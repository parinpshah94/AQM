  • Tableau (data Visualization and KPI dashboards made easy)
  • Optimizely (A/B Testing Tool)

Knowledge Snippet: If you dont already know this, A/B testing is a process that shows users different versions of a web page / app module and picks the winning page in terms of engagement using various statistical techniques. It is used by marketers and product designers globally to attract customers.
